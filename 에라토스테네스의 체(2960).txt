1) 문제 인식
N, K를 입력받고 에라토스테네스의 체 알고리즘을 이용해서 소수를 걸러낼 때,
K번째로 지워질 수를 출력하는 문제입니다. 

2) 문제 유형 파악
N 크기의 배열을 선언해서 배열 데이터를 이용해 에라토스테네스의 체 알고리즘을 이용하는데,
수를 지울 때마다 세서 K번째 지워지는 수를 찾아 출력하는 방식이 될 것입니다.

n, k 입력

배열 데이터 모두 1로 초기화

for (2부터 n까지 반복){

            arr[j] == 1이면 cnt++, arr[j] = 0;
            if (cnt == k이면) j 출력하고 종료
        }

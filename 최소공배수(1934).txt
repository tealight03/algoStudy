1) 문제 인식
테스트케이스의 수(n)를 입력받아 n번 만큼 두 정수 a, b를 입력받고,
두 수 a, b의 최소공배수를 출력하는 문제입니다. 

2) 문제 유형 파악
맨 처음 테스트케이스 수를 한번 입력 받고,
두 수 a, b를 N번 만큼 입력 받아 매번 두 수 최소 공배수를 출력합니다.
단순히 최소공배수를 구하는 공식을 사용하면 시간초과가 걸리므로
수업 시간에 다룬 "유클리드 호제법"을 이용합니다.

3) 알고리즘(슈도 코드)
---- gcd함수 ----
파라미터로 a, b 넘겨받아서

while (b가 0이 아닐 때 반복){
    remain <- a % b;
    a <- b;
    b <- remain;
}

---- main함수 ----
testcase, n1, n2, lcm 선언
testcase 입력받고

for (i = 0부터 시작해 testcase번만큼 반복){
    n1, n2 입력
    lcm <- n1 * n2 / gcd(n1, n2);
    lcm 출력
}

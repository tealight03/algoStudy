1) 문제 인식
길이가 최대 10^5인 숫자를 입력받아 이 숫자의 각 자릿수를 재조립했을 때
만들 수 있는 가장 큰 30의 배수를 구하는 문제입니다.
가능할 경우, 찾은 결과를 출력하고 불가능할 경우 -1을 출력합니다. 

2) 문제 유형 파악
길이가 최대 10^5이므로 문자열로 데이터를 입력 받아 오름차순으로 순서를 정렬하고, 
30의 배수라는 조건을 충족하는지 판단해야하므로 문자열에 0이 포함되어 있지 않은 경우 바로 -1을 출력하고, 
위 조건을 충족할 경우 모든 자릿수의 합이 3으로 나누어 떨어지는지 확인하여
나누어 떨어지지 않는 경우 -1을, 나누어 떨어진다면 num의 문자열을 합친 결과를 출력합니다.

3) 알고리즘(슈도 코드)
num = sorted(input(), reverse = True) # 입력 후 내림차순 정렬
total = 0

if '0' not in num:
    print(-1)
else:
    for i in num:
        total += int(i)
    
    if total % 3 != 0:
        print(-1)
    else:
        print("".join(num))
